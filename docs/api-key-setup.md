3. 输入新的API密钥（建议32位随机字符串）
4. 确认密钥设置

## 🔒 密钥安全建议

### 密钥生成建议
- 使用32位随机字符串
- 包含大小写字母、数字
- 避免使用易猜测的内容
- 示例：`aB3cD9eF2gH7iJ4kL8mN1oP5qR6sT3uV9wX2yZ7`

### 密钥保管
- 妥善保管密钥，不要泄露
- 不要将密钥提交到代码仓库
- 定期更换密钥
- 记录密钥设置时间
